# 4-Channel Encoder Motor Drive Module
![](https://github.com/YahboomTechnology/4-Channel-Motor-Drive-Module/blob/main/4-Channel-Encoder-Motor-Drive-Module.jpg)
# Introduction
This encoder motor drive module is a multifunctional motor drive expansion board designed for robots and smart cars. It supports simultaneous control of 4-channel DC motors and integrates encoder interface to monitor motor speed in real time for precise motion control. On board STM32F103RCT6 coprocessor to drive and obtain encoder data. It can communicate with the external master control through IIC or serial port to transmit data and receive drive instructions. Whether it is a DIY enthusiast, educational experiment, or electronic design-related competition, this expansion board can provide stable and efficient motor drive solution. Compatible with mainstream development platforms such as AR, Raspberry Pi, STM32, MSPM0, JetsonNANO, etc., can be easily integrated into your project.

* Note: Please select a suitable power supply (battery) according to the rated voltage of the motor you want to connect. Avoid excessive voltage and damage to the motor or driver board.

* If you want to connect 520 motor, please choose a 12.6V power supply.

* If you want to connect 310 motor or TT motor, please choose a 7.4V power supply.

# Features
1) 4-channel independent control: supports driving four DC motors at the same time, meets the needs of multi-motor collaborative work, and is suitable for complex robots or automation equipment.

2) Integrated encoder interface: built-in encoder signal processing function, can feedback motor speed to realize precise closed-loop control.

3) Built-in STM32F103RCT6 coprocessor: can drive the motor and obtains relevant data, effectively reducing the occupation of the master control pin resources.

4) Support serial port and IIC communication: can communicate with Arduino, Raspberry Pi, STM32, MSPM0, JetsonNANO, etc., transmit data and receive drive instructions.

5) Cleverly designed: anti-reverse connection interface layout and clear markings make it easy for users to quickly connect motors and sensors.
